# 📝 Smart To‑Do List App

## 📌 Overview
This is a simple yet elegant **To‑Do List application** built using **HTML, CSS, and JavaScript**.  
It helps users manage daily tasks with a clean and modern UI.

## ✨ Features
- ➕ Add new tasks easily  
- ✅ Mark tasks as completed  
- 🗑️ Delete tasks when not needed  
- 📊 View task statistics (completed vs pending)  
- 🕒 Task history section  
- 🎨 Stylish and responsive design  

## 🎯 Purpose
This project was created as part of my **InternPe Internship Task** to practice and improve my web development skills.  
It focuses on **UI/UX design** and **JavaScript functionality**.

## 🚀 How to Use
1. Clone or download the repository.  
2. Open `index.html` in your browser.  
3. Start adding tasks and manage them easily.
